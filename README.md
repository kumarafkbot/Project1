# Baithak — The Digital Chaupal

A social chat app with DMs, group chats, and a warli/village aesthetic.

## Deploy in 5 minutes (Railway)

1. Push this folder to a GitHub repository
2. Go to [railway.app](https://railway.app) and sign in with GitHub
3. Click **New Project → Deploy from GitHub repo** → select your repo
4. Railway auto-detects Node.js and deploys
5. Click **Settings → Generate Domain** to get your public URL

Everyone can now sign up and chat at your URL.

## Run locally

```bash
npm install
npm start
```

Open http://localhost:3000
